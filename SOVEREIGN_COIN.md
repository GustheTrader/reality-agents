## The Reputation System

SVRGN balance = reputation weight, not wealth.

```
HIGH BALANCE                    LOW BALANCE
─────────────                   ─────────────
More governance votes           Fewer votes
Higher prediction stakes        Lower stakes
Priority agent access           Basic access
Trusted mesh contributor        New contributor
```

**Not plutocracy. Meritocracy.** You earn SVRGN by contributing signal, not by buying in.

## Technical Implementation

### Option A: Simple Ledger (Phase 1)
```
- Internal database tracking
- No blockchain needed
- Fast, cheap, simple
- Good enough for 100-1000 members
```

### Option B: Token on Chain (Phase 2)
```
- ERC-20 or similar on L2 (Base, Arbitrum)
- Transparent, auditable
- Interoperable with DeFi
- Cross-instance transferable
```

### Option C: Sovereign Chain (Phase 3)
```
- Custom blockchain for Wrong Room Collective
- Optimized for prediction markets
- Privacy-preserving
- Fully decentralized
```

**Phase 1 starts now.** Simple ledger. No crypto complexity. Just functional currency.
