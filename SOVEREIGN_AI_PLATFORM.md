## The Mesh Network

Instances don't share DATA. They share **INSIGHTS.**

```
SMB Instance A                  Member Instance B
├── Learns from own data        ├── Learns from own data
├── Generates insights          ├── Generates insights
└── Publishes to mesh ──────────└── Receives mesh insights
        │                               │
        └───────────┬───────────────────┘
                    │
            ┌───────┴───────┐
            │  MESH LAYER   │
            │ • Anonymized  │
            │ • Aggregated  │
            │ • Voluntary   │
            │ • Auditable   │
            └───────────────┘
```

**What gets shared:**
- Aggregated patterns (not individual data)
- Market predictions (NOESIS)
- Research findings
- Tool recommendations
- Strategy templates

**What NEVER gets shared:**
- Your private data
- Your customer information
- Your financials
- Your personal context
- Anything you don't explicitly publish
